SendJSON Function options

This function will make an ajax sends data to the page of your choice with the "TargetLink" you gave him. Alternatively, you can overload this function if you want to make another use.

Variable

Default

function (json) {
        $.ajax({
          type: "POST",
          url: plugin.tdpersonnalised.link,
          data: json,
        }).success(function(){
          plugin.tdpersonnalised.SuccesSend();
          resetChange();
        }).error(function(){
          plugin.tdpersonnalised.ErrorSend();
        });
}

Possible Values

Any function

You can make the function of your choice, you can use the variable "json" which contains the data to send. To see an example of function overloading you can see those of dataDB page or the examples below.
