Main Buttons Div options

Variable

mainButtonDiv

Default

#mainButtonDiv

Possible Values

class (not recommended) or id or even CSS path

You can leave the default value but if you use several times the plugin on the page you will be obliged to change the value. You can use a class or id, but I do not recommend the class to avoid repetition and to generate functioning wrong.

<div id="MyTable">
  <div id="maindivbutton"></div>
  <table class="table">
    <thead>
      <tr>
        <th>First Name</th>
        <th>Last Name</th>
        <th>Username</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Vincent</td>
        <td>Riouallon</td>
        <td>@VRiouallon</td>
      </tr>
    </tbody>
  </table>
</div>
$('#MyTable').TextToInput({
    mainButtonDiv  : "#maindivbutton"
});
First Name Last Name Username
Vincent Riouallon @VRiouallon
