Buttons's class options

Variable

buttunClass

Default

btn

Possible Values

String (representing class)

The variable "buttonClass" is the name of the class for formatting tag button in that we want. Using bootstrap it comes to the class "btn". That is why it is the one I use.

Variable

modifiedclass,saveclass,cancelclass,ToSaveclass

Default

"btn-primary","btn-success","btn-danger","btn-success"

Possible Values

String (representing class)

The four above variables to customize the buttons as desired. Again I use the bootstrap class, but you can put your own.

$('#myTarget').TextToInput({
    buttunClass  : "btn",
    modifiedclass  : "btn-primary",
    saveclass  : "btn-success",
    cancelclass  : "btn-danger",
    ToSaveclass : "btn-success"
});
