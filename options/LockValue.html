Lock Values options

Variable

notChange

Default

.lockValue

Possible Values

class or id (not recommended) or even CSS path

You can leave the default value no matter how many times you use the plugin on the same page. You can use this class to the fields that must remain unchanged.
The class "actionCell" (the default value of "actionCell") is considered as a LockValue. For more information on the class "actionCell" please visit the page: "Type of control".
Warning: I reminds you that javascript is a client language, it is not secure, remember to check / ignore some values if you made inclusion in a database.
