Title Buttons options

Variable

modifiedtitle, savetitle, canceltitle, ToSavetitle

Default

"","","",""

Possible Values

String

By default no title is on the buttons.
You can add if you wish as I do below. 

<div id="basicTable">
  <div></div>
  <table class="table">
      <thead>
        <tr>
          <th>First Name</th>
          <th>Last Name</th>
          <th>Username</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>Vincent</td>
          <td>Riouallon</td>
          <td>@VRiouallon</td>
        </tr>
        <tr>
          <td>Rémi</td>
          <td>Collignon</td>
          <td>Miton</td>
        </tr>
        <tr>
          <td>Gilles</td>
          <td>Biannic</td>
          <td>GillouBZH</td>
        </tr>
      </tbody>
    </table>
</div>
$('#basicTable').TextToInput({
    mainButtonDiv : "#basicTable div:nth-child(1)",
    modifiedtitle : "Change",
    savetitle : "Let&apos;s save",//I replaced the apostrophe by the html code "&apos;" to avoid conflict, otherwise your title will stop at the first '
    canceltitle : "No I have change my mind",
    ToSavetitle : "Aaaand let&apos;s validate"
});
