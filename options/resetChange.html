resetChange Function informations

This function is called when you want to do a reset of the plugin values (parameters are not changed). This function allows to continue sending new values after a first sends already completed.

Function

resetChange()

Possible Values

None. This function can not be overloaded.

You can see examples of using the "resetChange" function on these three pages: sendfunction, SuccessSend, ErrorSend
