TextToInput API

TextToInput is a simple jQuery plugin for the processing of data by table or list. It is completely customizable from its options or functions that are rewritable by the user. You can refer to the documentation of options and features for more details.

Remember, this is how you instantiate the plugin:

Copy
/*If we use all default values*/
  $('#basicTable').TextToInput();
