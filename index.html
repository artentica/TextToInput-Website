TextToInput is an easiest way to manage database from a web page or personnalised informations. The plugin is written so as to be as customizable as possible.

This plugin is designed to send only the minimum information necessary to update the database or changes.

$('.example').TextToInput();

This plugin requires at jquery 1.7 or higher
