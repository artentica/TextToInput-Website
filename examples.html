Getting Started

The first thing you'll need to do is download TextToInput. And then include Jquery and TextToInput's script.

Copy
<!-- To include in your code --> 
<script src="/path/to/jquery.min.js" type="text/javascript"></script> <!--At least jquery 1.7 -->
<script src="/path/to/texttoinput.min.js" type="text/javascript"></script>

The rest of this guide and any code snippets on this site, unless otherwise noted, will be using the TextToInput's V2.0.

For the example I will use bootstrap CSS to show you how you can apply style on the different part of the plug-in.

For some functions I will also use bootstrap JS for the same reason than precedently, i will personnalised some function.

Where insert code

Copy
<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="Content-Type" content="text/html" charset="utf-8" />
    <title> <!-- Your Title --> </title>
    <link rel="stylesheet" type="text/css" media="screen" href="css/sryle.css" />  <!-- Your CSS Code Here --> 

    <!--[if IE]>
    <script src="http://html5shiv.googlecode.com/svn/trunk/html5.js"></script>
    <![endif]-->
  </head>
  <body>

     <!-- Your HTML Code Here --> 

  </body>
